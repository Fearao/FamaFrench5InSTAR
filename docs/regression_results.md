# 因子回归结果

## 五因子 Benchmark（Equal-Weight 组合）
- 样本：2019-08-09 ~ 2025-12-05，共 301 个周度观测；自变量改为“5 分位分组 top vs. bottom”的因子收益，被解释变量为等权组合超额收益。
- R² / 调整 R²：0.956 / 0.955；HAC(maxlags=4) 纠偏标准误。

| 因子 | 系数 | t 值 | p 值 |
| --- | --- | --- | --- |
| α | -6.37e-04 | -1.23 | 0.218 |
| MKT-RF | 1.057 | 60.73 | <1e-6 |
| SMB | 0.130 | 11.56 | <1e-6 |
| HML | 0.0956 | 5.23 | 1.66e-07 |
| RMW | 0.0164 | 1.49 | 0.136 |
| CMA | 1.21e-04 | 0.01 | 0.991 |

> 解释：切换到“5 组 top-bottom”因子后，模型仍能解释 95% 以上波动；市场暴露约 1.06，规模因子转为中等正值；HML 变得显著为正，而 CMA 在该样本中几乎不起作用。

## 五因子 + 动量（Equal-Weight 组合）
- 同样使用 301 个周度观测，并加入 `MOM`；R² / 调整 R²：0.956 / 0.955，与五因子模型几乎相同。

| 因子 | 系数 | t 值 | p 值 |
| --- | --- | --- | --- |
| α | -6.17e-04 | -1.19 | 0.234 |
| MKT-RF | 1.057 | 61.18 | <1e-6 |
| SMB | 0.130 | 11.60 | <1e-6 |
| HML | 0.0963 | 5.37 | 7.79e-08 |
| RMW | 0.0165 | 1.49 | 0.135 |
| CMA | 2.74e-04 | 0.02 | 0.980 |
| MOM | 0.0016 | 0.23 | 0.819 |

> 引入动量因子后，R² 提升小于 0.00002，`MOM` 系数也不显著；说明在等权组合+5 组分组体系下，动量对解释力贡献依旧有限。
