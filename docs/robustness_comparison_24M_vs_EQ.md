# 第6章：表现-稳健性权衡分析

**权衡框架**：

- **牺牲成本**：ΔR²由3.22pp降至2.05pp（-1.17pp）。

- **稳健性收益**：

  - 时间稳定性改善22.5pp，约等于89%修复幅度；系数SD从+54.2%转为-3.1% 。

  - 趋势一致性：时间相关系数提升+0.55，极差/标准差/CV分别改善-17.5pp、-8.14pp、-0.23。

  - 类型稳健性：size_bm +36.0pp、size_inv +8.0pp、size_op +13.0pp。

**经济学解释**：

1. 多窗口分散化效应：等权组合缓冲单一24M窗口的样本特异波动，类似投资组合理论中的风险分散。

2. 信号频率分散：短/长窗口在不同市场状态下交替贡献，降低由噪音或拥挤导致的单点失灵。

3. 类型异质性对冲：不同窗口在size_bm、size_inv、size_op中的敏感度互补，对冲了某一类型崩溃带来的系统性拖累。

**实务建议**：

- 若追求最大R²，可继续使用MOM_24M但需接受时间与类型层面的高回撤风险。

- 若追求稳健性，应切换到MOM_EQ并监控其在新增样本期的表现；可进一步探索非等权的动态窗口。

- 综合推荐：MOM_EQ的稳健性收益远超1.17pp的ΔR²损失，应成为新的阶段2最优因子。
