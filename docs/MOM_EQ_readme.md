# MOM_EQ 回归结果说明

该文档对应 `docs/MOM_EQ_coeffs.csv`，记录了六因子（五因子 + 等权动量 MOM_EQ）回归的全部系数、t 值与 p 值。核心指标如下：

| 项目 | 系数 | t 值 | p 值 | 解读 |
| --- | --- | --- | --- | --- |
| α | -1.36e-03 | -2.74 | 0.006 | 等权组合存在轻微负 α，说明在控制因子后仍有约 -13bps 的周度残差，但幅度较小。 |
| MKT-RF | 1.050 | 57.19 | <1e-6 | 市场 β≈1.05，组合几乎完全跟随市场波动。 |
| SMB | 0.114 | 10.84 | 2.3e-27 | 明显的小盘暴露，等权组合对小市值股票更敏感。 |
| HML | 0.068 | 5.09 | 3.5e-07 | 略偏向高账面市值比（价值型）。 |
| RMW | 0.0156 | 1.57 | 0.117 | 盈利因子正相关但只在 10% 附近，解释力有限。 |
| CMA | -0.0050 | -0.53 | 0.595 | 投资因子基本无贡献。 |
| MOM_EQ | -0.1265 | -4.07 | 4.7e-05 | 等权动量（4/8/16/26/52/78 周平均）呈显著负暴露，说明组合收益与动量高低相关：动量高的股票在该组合中往往出现回吐。 |

配合 `docs/MOM_EQ_meta.json` 可获取样本数（n=299/300，视窗口有效期而定）、R²≈0.961、HAC(4) 设定等信息；更完整的各窗口比较见 `docs/momentum_comparison.md`。
